# htb-offensive-security-labs

Educational repository containing scripts, notes and automation developed during structured Offensive Security training on Hack The Box Academy.

The purpose of this repository is to document hands-on learning, experimentation and methodology development while transitioning toward professional penetration testing and offensive security roles.

---

## Training Paths

Current learning paths:

- **CREST CCT INF Preparation**
- **Senior Web Penetration Tester**
- Advanced Web Application Security modules

Additional paths will be added throughout the Hack The Box Academy subscription period.

---

## Repository Goals

This repository focuses on:

- Understanding attack methodologies
- Practicing real-world penetration testing workflows
- Reinforcing exploitation concepts
- Automating repetitive assessment tasks
- Building a structured offensive security mindset

Scripts included here are intentionally simple and educational.  
They are designed to support learning and methodology comprehension rather than replace professional security tools.

---

## Repository Structure

- web/ → Web application security labs (SQLi, XSS, auth bypass, etc.)
- enumeration/ → Reconnaissance and information gathering helpers
- privilege-escalation/ → Local privilege escalation practice
- automation/ → Custom workflow automation scripts
- notes/ → Study notes and methodology references
